Структура проекта:

assets (Ресурсы игры: текстуры, 3d модели, звуки и прочее. Эта папка не включена в репозитарий.)
external (Внешние библиотеки для загрузки ресурсов, проигрывания звуков и прочее. dll, lib и заголовочные файлы.)
src (Исходники игры.)
	lib (Базовые классы для всей программы.)
	core (Игровая логика. Инициализация игры, обработка ввода.)
	engine (Движок игры: перемещения, соударения и прочая физика.)
	model (Игровые обьекты и сопутствующие данные.)
	render (Логика отрисовки игрового процесса на экране.)
		ui (Логика отрисовки игровых экранов.)
	ui (Пользовательский интерфейс. Игровые экраны и логика переключения между ними.)
